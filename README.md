## ⚙️ Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/ricardocardoso90/react-project-my-portfolio.git
```

Acesse a pasta do projeto:

```bash
cd react-project-my-portfolio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
